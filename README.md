# Hi, I'm Prasad Jawale

I'm a Master's student in Data Science at the University of Wisconsin–Madison, driven by a passion for uncovering insights from data and building intelligent systems that solve real-world problems. I like working with data, transforming and analyzing it to help solve real-life problems better. My area of interest is in combining web software development with artificial intelligence and data science to create enterprise grade applications.

## Technical Focus 
- Designing and building deep learning models with PyTorch, specializing in Natural Language Processing applications, including topic modeling and sentiment analysis.
- Developing traditional machine learning solutions for forecasting, classification, and regression tasks in business and financial domains.
- Implementing end-to-end AWS pipelines for ML model deployment and scalable production workflows.
- Full-stack web development using ReactJS for frontend and FastAPI for backend services.
- Native Android application development in Java/Flutter.
