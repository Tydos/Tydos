# Hi, I'm Prasad Jawale

I'm a Master's student in Data Science at the University of Wisconsin–Madison, driven by a passion for uncovering insights from data and building intelligent systems that solve real-world problems. I like working with data, transforming and analyzing it to help solve real-life problems better. My area of interest is in combining web software development with artificial intelligence and data science to create enterprise grade applications.

## About Me

My background blends statistics, machine learning, and hands-on experience with manipulating both structured and unstructured data using SQL and NoSQL databases. I have done software developement in both Android and Web based applications and integrated ML aspects into it. I'm currently strengthening my foundations through coursework in Machine Learning, Statistical Methods & Modeling, Optimization, and Natural Language Processing

## Technical Focus 
- Building and training deep learning models with **TensorFlow** and **PyTorch**, especially for **Computer Vision** and **Natural Language Processing**
- Applying traditional machine learning methods to develop **forecasting**, **classification**, and **regression** models for business and finance  
- Experienced in data preprocessing, feature engineering, and model evaluation using **Pandas** and **R**.
- Full stack web development using **React and Python**
- Native Android development in **Android Studio** using **Java**
